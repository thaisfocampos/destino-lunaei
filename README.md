# Destino Lunaei — Landing Page

Site oficial do **Mapa Natal da Vida** da Destino Lunaei.

## Estrutura

```
/
├── index.html       → Landing page principal
├── netlify.toml     → Configuração do deploy no Netlify
└── README.md        → Este arquivo
```

## Deploy

O site é hospedado no **Netlify** com deploy automático a partir deste repositório.

Toda vez que um commit é feito na branch `main`, o Netlify detecta a mudança e publica a nova versão automaticamente em até 1 minuto.

## Como atualizar o site

1. Edite o arquivo `index.html`
2. Faça commit e push para a branch `main`
3. O Netlify publica automaticamente

```bash
git add .
git commit -m "Atualiza landing page"
git push origin main
```

## Tecnologias

- HTML + CSS puro (sem dependências)
- Fontes: Google Fonts (Cormorant Garamond, Cinzel, Lato)
- Hospedagem: Netlify
- Checkout: Kiwify
